# 口腔医院预约平台小程序

#### 介绍
口腔医院预约平台小程序是基于uni-app+uView+HBuilder X开发的预约类小程序，目前小程序端还没有对接后端，只是一个单纯的演示项目

#### 软件架构
##### 小程序端

- 开发工具：HBuilder X
- UI框架：uView

- 首页

  - 医师团队
